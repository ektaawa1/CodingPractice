## âœ… 2. **addToHead(DoublyLinkedNode node)**

ğŸ” **Purpose**: Insert `node` **right after dummy head** (marking it "most recently used").

### Before:

```
head <-> firstNode
```

### Code:

```java
node.next = head.next;   // node points to firstNode
node.prev = head;        // node points back to head
head.next.prev = node;   // firstNode's prev now points to new node
head.next = node;        // head now points to new node
```

### After:

```
head <-> node <-> firstNode
```

âœ”ï¸ New node inserted immediately after `head`.


### Visually:

```

```

After removal:
